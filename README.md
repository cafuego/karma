Slack - Karma
=============

A simple karma bot for Slack.

## Setup

*Binary*

*Container*

## Usage

*Give karma*

```
karma nickschuch++
```

```
karma nickschuch+=10
```

*Take karma*

```
karma nickschuch--
```

```
karma nickschuch-=10
```

*Check karma my karma (if matches username)*

```
karma
```

*Check karma of others*

```
karma nickschuch
```
